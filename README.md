# Chess Game OOP

This project implements a chess game using object-oriented programming in C++.

## Structure

- `include/` : Header files for pieces, board, and game logic
- `src/` : Source files implementing the chess game
- `output/` : Compiled executable and object files
- `screenshots/` : Project report screenshots

## Build

Use `make` to compile the project:

    make

Or compile manually with a C++17 compiler and `-Iinclude`.
